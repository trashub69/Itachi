Valentine Page Template (GitHub Pages ready)

1) Put your background image here:
   ./assets/itachi-bg.jpg
   and set the path in css/style.css (variable --bg-url)

2) Put your audio files here:
   ./assets/crow.mp3
   ./assets/music.mp3

3) Open locally:
   - Double click index.html
   OR
   - python3 -m http.server 8080
     then open http://localhost:8080

4) GitHub Pages:
   - Create repo, upload everything
   - Settings -> Pages -> Deploy from branch (main / root)
